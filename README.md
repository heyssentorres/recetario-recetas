# Recetario de Recetas

Web application that consumes a public API to display food recipes organized by categories.

## Features
- Display recipes obtained from an external API
- Recipes organized by categories
- Dynamic content loading
- Simple and responsive interface

## Technologies
- HTML
- CSS
- JavaScript
- REST API

## How to Run
1. Clone the repository
2. Open the project folder
3. Run the project by opening `index.html` in your browser.
